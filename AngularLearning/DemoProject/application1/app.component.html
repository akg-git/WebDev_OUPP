What is Module?

Component Creation Process

Step 1
command: ng g c component1
Step 2
Write something inside "component1.component.html"
Step 3
Copy the selector from "component1.component.ts" file and paste into "app.component.html"
Step 4
Run your server using command "ng serve" and run on browser using "localhost:4200"
